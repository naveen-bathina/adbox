# User Flows

## User App (Flutter)

1. Onboarding & Signup
   - Welcome screen
   - Signup (age, gender, location, phone/email)
   - Accept privacy/terms, confirm age
2. Home Screen
   - List available ads/surveys
   - Show wallet balance
3. Ad & Survey Flow
   - Tap ad → view full ad
   - Survey unlocks → answer questions
   - Points awarded, wallet updated
4. Wallet
   - View balance, transaction history
   - Request withdrawal (enter UPI ID if needed)
5. Notifications
   - Points earned, withdrawal status
6. Support & Settings
   - FAQ, contact support, edit profile

## Admin Panel (React)

1. Login
2. Dashboard (metrics)
3. Ad Management (upload, edit, targeting)
4. Survey Management (create, link, set rewards)
5. User Management (view/search, transactions)
6. Payout Management (review/approve withdrawals)
7. Analytics & Export
8. Support Tools (tickets)

## Notes

- All flows require secure auth and validation
- UX should be mobile-first, accessible, and responsive
